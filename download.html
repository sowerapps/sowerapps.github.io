Packages without '+' in the file name contain only the applications. Packages with '+' in the file name contain the applications, and a minimal module library. None of the packages contain ThML files such as ThML Viewer uses, for ThML files visit the Christian Classics Ethereal Library. All packages are 64 bit. And all are completely free.

Sower Applications-1.3-Mac-Monterey+.dmg

The module library is here. The modules are compressed with bzip2, and will have to be decompressed. Sower Module Installer can be used to decompress, and install the modules.

If you need to build Sower Applications for your system the source is here.
