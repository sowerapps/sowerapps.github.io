Packages with 'basic' in the file name contain only the applications. Packages with 'full' in the file name contain the applications, and the current module library. None of the packages contain ThML files such as ThML Viewer uses, for ThML files visit the Christian Classics Ethereal Library. For MSW there are 32 bit, and 64 bit versions. All Linux packages are 64 bit.

The module library is located here. All modules in the library are included in the 'full' packages.


MSW

sowerapplications1.0-32b-setup-basic.exe
sowerapplications1.0-32b-setup-full.exe
sowerapplications1.0-64b-setup-basic.exe
sowerapplications1.0-64b-setup-full.exe

* If the applications fail to start from the menu set the working directory to '/usr/bin'.
