Packages without '+' in the file name contain only the applications. Packages with '+' in the file name contain the applications, and a minimal module library. None of the packages contain ThML files such as ThML Viewer uses, for ThML files visit the Christian Classics Ethereal Library. All packages are 64 bit. And all are completely free.

Mac
DMG's that I have built for Mac fail to work after uploading to github, and downloading onto a Mac. So, for now Sower Applications will have to be built from source on Mac's.

The module library is here. The modules are compressed with bzip2, and will have to be decompressed. Sower Module Installer can be used to decompress, and install the modules. The library is small. I do not intend to maintain a large library. You can find many books at CCEL.org, and gutenberg.org. You can create a library for ThML Viewer with ThML files from CCEL, or you can use Sower Module Writer to create modules for Sower. You can use Sower Librarian to maintain an entire library of modules built from ThML files. You can also use Sower Project Writer, and ThML Builder to create ThML files from epub books found at Project Gutenberg, and many other other sites.

If you need to build Sower Applications for your system the source is here.
