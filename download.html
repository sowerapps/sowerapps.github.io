Download

Packages with 'basic' in the file name contain only the applications. Packages with 'full' in the file name contain the applications, and the current module library. None of the packages contain ThML files such as ThML Viewer uses, for ThML files visit the Christian Classics Ethereal Library. For MSW there are 32 bit, and 64 bit versions. All Linux packages are 64 bit. And all are completely free.

If you have downloaded only the basic package, or are looking for new modules you can visit the module library here to download modules for the software.

sowerapplications1.0-32b-setup-basic.exe
sowerapplications1.0-32b-setup-full.exe
sowerapplications1.0-64b-setup-basic.exe
